# تجديد مطابخ مكة

نسخة مستقلة وآمنة من صفحة هبوط تجديد المطابخ، جاهزة للنشر كموقع static.

## الملفات

- `index.html`: صفحة الهبوط.
- `assets/`: ملفات التنسيق والتفاعل المحلية.
- `images/`: صور الصفحة مستضافة محليًا داخل الريبو.
- `vercel.json`: المسارات ورؤوس الحماية.

لا تحتاج الصفحة إلى تثبيت حزم أو تنفيذ أمر build.
